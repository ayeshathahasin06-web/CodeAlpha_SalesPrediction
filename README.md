# 📈 Sales Prediction using Machine Learning

## 📌 Project Overview

This project predicts product sales based on advertising expenditure on TV, Radio, and Newspaper using Machine Learning. A Linear Regression model is trained to analyze the relationship between advertising budgets and sales, helping businesses estimate future sales based on marketing investments.

## 🎯 Objective

To build a machine learning model that accurately predicts sales using advertising data and evaluates its performance using regression metrics.

## 📂 Dataset

- Dataset: Advertising.csv
- Features:
  - TV Advertising Budget
  - Radio Advertising Budget
  - Newspaper Advertising Budget
- Target:
  - Sales

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 🤖 Machine Learning Model

- Linear Regression

## 📊 Project Workflow

- Import required libraries
- Load and explore the dataset
- Check dataset information
- Handle missing values
- Perform statistical analysis
- Visualize advertising vs. sales using graphs
- Split the dataset into training and testing sets
- Train the Linear Regression model
- Evaluate the model using MAE, MSE, RMSE, and R² Score
- Predict sales for new advertising values

## 📈 Model Performance

- Mean Absolute Error (MAE): 1.4608
- Mean Squared Error (MSE): 3.1741
- Root Mean Squared Error (RMSE): 1.7816
- R² Score: 0.8994

## 📷 Visualizations

- TV Advertising vs Sales
- Radio Advertising vs Sales
- Newspaper Advertising vs Sales
- Actual Sales vs Predicted Sales

## ✅ Conclusion

The Linear Regression model successfully predicts sales based on advertising expenditure. The project demonstrates the complete machine learning workflow, including data preprocessing, visualization, model training, evaluation, and prediction. The achieved R² Score of approximately 0.90 indicates strong predictive performance.
